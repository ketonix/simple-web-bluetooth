Example Ketonix using Bluetooth

You need to use the Chrome browser. It can be downloaded from here

Enable the Chrome flag "Use the new permissions backend for Web Bluetooth".


1. Enter "chrome://flags" in Chrome URL field and hit enter.
2. Search for "bluetooth" in flags
3. Enable "Use the new permissions backend for Web Bluetooth"
4. Relaunch Chrome to make the setting effective.
